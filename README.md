## 🚀 Run Project

1️⃣ Clone project and access its folder:

```bash
$ git clone https://github.com/jtiagosantos/arloesi-task.git
$ cd arloesi-task
```

2️⃣ Install dependencies:

```bash
$ yarn or npm i
```

3️⃣ Run tests:

```bash
$ yarn test or npm run test
```

4️⃣ Start project:

```bash
$ yarn dev or npm run dev
```
